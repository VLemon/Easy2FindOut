# Easy2FindOut

最近有些想法，想记录下来，就建立了这个repo，主要用来记录日常中最熟悉却又最陌生的一些东西，
通过一些方法找出原理层面的东西，争取简单容易理解。

主要目录
1. 内存管理 （ARC的实现，autorelease，weak）
2. Block
3. runtime  (KVO, MJExtension)
4. runloop  (NSTimer)
5. GCD
6. 事件传递 （自动化埋点）
7. 源码解析 上 （SDWebImage）
8. 源码解析 下 （AFNetworking + HTTP / HTTPS /HTTP2）
9. 消息机制 （NSProxy）
10. App启动那些事 (Mach-O, 动态连接器,FishHook) 
11. App性能监控以及优化
12. CoreML
